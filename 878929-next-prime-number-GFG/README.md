# [Next Prime Number](https://www.geeksforgeeks.org/problems/next-prime-number--163011/1?page=1&category=Java&difficulty=Medium&sortBy=latest)
## Medium
Given an integer&nbsp;n.&nbsp;Write a program to find the first prime number greater than n.
Examples:  
Input: n = 15
Output: 17
Explanation: 17 is next prime number.
Input: n = 7
Output: 11
Explanation: 11 is the prime number next to 7.
Constraints:1 &lt;= n&nbsp;&lt;= 500
